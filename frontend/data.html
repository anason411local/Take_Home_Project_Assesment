Understanding the Metrics
MAPEMean Absolute Percentage Error

Measures average prediction error as a percentage. Lower is better. Target: <20%

MAEMean Absolute Error

Average absolute difference between predicted and actual values in dollars. Lower is better.

RMSERoot Mean Square Error

Similar to MAE but penalizes larger errors more. Useful for detecting outlier predictions.
